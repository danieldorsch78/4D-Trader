Privacy Policy

4D Market Intelligence — developed by Daniel Dorsch / 4D Digital Solutions

Effective date: March 13, 2026

1. Overview

4D Market Intelligence ("the App") is a financial market analysis and decision-support tool. This policy describes how the App collects, uses, and protects your data.

2. Data We Collect

Data stored locally on your device only:

  • Watchlists & Preferences — Your custom watchlists, alert rules, display settings, and selected markets are stored on-device using a local database.
  • API Keys — If you provide free API keys (Finnhub, Alpha Vantage, Brapi), they are stored encrypted on your device only. They are never transmitted to our servers.
  • Cached Market Data — Recent price quotes and historical data are cached locally to reduce network usage.

Data transmitted to third-party services:

  • The App fetches market data from public financial APIs (CoinGecko, Finnhub, Alpha Vantage, Brapi). Your device's IP address is visible to these services during requests. We do not control or receive this data.

Data we do NOT collect:

  • No personal information (name, email, phone, location)
  • No analytics or tracking (no Firebase Analytics, no ad SDKs)
  • No account registration required
  • No data sent to our servers — the App has no backend

3. Third-Party Services

The App communicates with these external APIs for market data:

  • CoinGecko API — Cryptocurrency data (CoinGecko Privacy Policy)
  • Finnhub API — Stock quotes and symbol search (Finnhub Terms)
  • Brapi API — Brazilian market data (Brapi Terms)

No user data is shared with these services beyond what is technically required to make API requests (IP address, API key if provided).

4. Data Storage & Security

  • All user data is stored locally on your Android device.
  • API keys are stored using Android's encrypted SharedPreferences.
  • Network communication uses HTTPS exclusively (cleartext traffic is disabled).
  • Uninstalling the App deletes all local data.

5. Children's Privacy

The App is not directed at children under 13. We do not knowingly collect data from children.

6. Changes to This Policy

We may update this policy from time to time. Changes will be posted on this page with an updated effective date.
